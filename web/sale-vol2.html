SYSTEM // ANALYSIS

ลูกค้าตัดสินใจเลื่อนผ่านโพสต์ของคุณ
ภายใน 0.3 วินาที — ก่อนสมองส่วนคิดจะตื่น

งานวิจัยจาก Microsoft Research พบว่ามนุษย์ในยุคดิจิทัลมีช่วงความสนใจเฉลี่ย 8 วินาที — น้อยกว่าปลาทอง

แต่ตัวเลขที่น่ากลัวกว่าคือ — การตัดสินใจ “หยุด” หรือ “เลื่อนผ่าน” เกิดขึ้นภายใน 0.3 วินาทีแรก โดยสมองส่วน Limbic ที่ไม่ผ่านความคิด

สมองยุคหินถูกออกแบบมาเพื่อสแกนทุ่งหญ้าหา “เสือ” กับ “ผลเบอร์รี่” — ตอนนี้มันต้องประมวลผล Feed ที่ยาวเป็นไม่รู้จบ จึงสร้าง กำแพง อัตโนมัติ Skip ทุกอย่างที่ไม่จำเป็นต่อการอยู่รอด // from Chapter 02: The Scroll

Dopamine Loop — ทำไม Feed ถึงเสพติด? เพราะสมองหลั่ง Dopamine ตอน “คาดหวัง” ไม่ใช่ตอน “ได้รับ” → ทุกการ scroll คือลุ้นว่าจะเจอ “รางวัล”

Pattern Interrupt — วิธีเดียวที่จะ “หยุด” นิ้วของลูกค้าคือ “หัก Pattern” ของ Feed — ไม่ใช่ทำให้สวยกว่า แต่ทำให้ต่างจากสิ่งที่อยู่รอบข้าง

Memory Encoding — ลูกค้าจะจำคุณได้ก็ต่อเมื่อสมองตัดสินใจว่า “คุ้มค่ากับการจำ” — ซึ่งมีกฎเฉพาะที่ไม่มี Marketing Course ไหนสอน

Vol.1 สอนว่า ทำไมคนซื้อ — แต่ Vol.2 สอนว่า ทำอย่างไรให้คนเห็นคุณ ก่อนเลื่อนผ่าน

เพราะในยุคดิจิทัล — Content ดี + ไม่มีใครเห็น = ไม่มีอยู่จริง
และ ความสนใจ คือสกุลเงินที่หายากที่สุดของศตวรรษนี้
