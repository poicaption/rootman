FORMULA

สมการ Hook 4 แบบที่ทำงาน

HOOK = SPECIFIC + CONFLICT + STAKE

1. ตัวเลขเฉพาะ — "200 ออเดอร์" ดีกว่า "เยอะมาก"
2. ความขัดแย้ง — "ขายเยอะแต่ไม่มีเซล" ทำให้สมองต้อง pause
3. ความเสี่ยง — "ถ้าไม่อ่านวันนี้คุณจะ..."
4. คำถามที่กวนใจ — "ทำไม X ที่ดีกว่ากลับไม่ขาย?"

ราคา คือเรื่องเล่า

เป้าหมาย: เข้าใจว่าราคาไม่ใช่ตัวเลข — มันคือเรื่องเล่าที่สมองตีความ และคุณวางบริบทอย่างไร เปลี่ยน "แพง" ให้เป็น "คุ้ม" ได้

FRAMING

เรื่องเล่าเปลี่ยนราคา 100%

แบบ A: "กาแฟแก้วละ 120 บาท" — แพง

บทที่ 04 4 นาที

สถาปัตยกรรมตัวเลือก

เป้าหมาย: ออกแบบหน้าขาย / ฟอร์ม / เมนู ให้ลูกค้าเลือกสิ่งที่คุณอยากให้เลือก โดยไม่รู้สึกถูกบังคับ
PRINCIPLE

3 หลัก Choice Architecture

1. Default — ค่าเริ่มต้นที่ติ๊กไว้แล้ว = ผู้ใช้ 80% จะปล่อยไว้ (Netflix auto-renew, donation opt-in)

2. Decoy — ตัวเลือกที่ดูเลวกว่า เพื่อทำให้ตัวที่คุณอยากให้เลือกดูดีขึ้น

3. Limit — ตัวเลือกมากเกิน = สมอง freeze (Hick's Law) — 3-5 ตัวคือจุดที่ดีที่สุด

บทที่ 05 4 นาที

กับดักอัตลักษณ์

เป้าหมาย: เข้าใจว่าลูกค้าไม่ได้ซื้อสินค้า — เขาซื้อ "ตัวตนที่อยากเป็น" และแบรนด์ที่เก่งคือคนทำให้เขา เห็นตัวเองในกระจกของแบรนด์
MIRROR · PRINCIPLE

"คนแบบเรา" vs "ลูกค้าทั่วไป"

คนเชื่อ คนแบบเขา มากกว่า expert — ความสำเร็จของ Apple, Patagonia, Tesla คือทำให้ลูกค้ารู้สึกว่า "ฉันคือคนแบบนี้"

// แบรนด์ทั่วไป

"ซื้อสินค้าเรา ของดี ราคาถูก"

ลูกค้าได้ยินทุกวัน — สมอง filter ออกทันที ไม่มีตัวตน ไม่มีฝ่าย

// แบรนด์ที่ชนะ

"คนทำงานสร้างสรรค์ ใช้ของชิ้นนี้"

ลูกค้ามอง — แล้วถามตัวเอง "ฉันใช่คนแบบนั้นไหม?" → ซื้อ เพื่อยืนยันตัวตน

FORMULA · IDENTITY LOOP

วงจร 3 ขั้น

SEE (เห็นคนแบบนั้น) → FEEL (รู้สึกอยากเป็น) → BUY (ซื้อเพื่อพิสูจน์)

หน้าที่ของคุณ: ทำให้ลูกค้าเห็น "คนแบบเขา" ใช้สินค้าอย่างเป็นธรรมชาติ — ไม่ใช่ดารา ไม่ใช่ ad คนรับจ้างถ่าย

// KEY TAKEAWAY
  • ลูกค้าไม่ซื้อ "สินค้าที่ดีที่สุด" — ซื้อ "ตัวตนที่ใกล้ที่สุด"
  • "คนแบบฉัน" > "คนดัง" — ความเฉพาะ > ความใหญ่
  • ระวัง Cult Trap: ตัวตนแคบเกิน = ตลาดเล็ก / ตัวตนกว้างเกิน = ไม่มีใคร

บทที่ 07 4 นาที

สมการ ไวรัล

เป้าหมาย: เข้าใจว่าทำไมคอนเทนต์บางตัว "ระเบิด" และบางตัวเงียบ — แล้วเขียนคอนเทนต์ที่มีโอกาสไวรัลสูงขึ้นโดยไม่ใช่บังเอิญ
FORMULA

K-Factor — สมการการแพร่

K = i × c
i = invitations sent / user · c = conversion rate

ถ้า K > 1 → คอนเทนต์ ทบทวีตัวเอง (ไวรัล) — ถ้า K < 1 → ตายลงเรื่อย ๆ

ตัวอย่าง: คน 100 ดู → 30 แชร์ → แชร์ละเข้าถึง 5 คนใหม่ → K = 1.5 ✓

DRIVERS · 6

6 ตัวขับเคลื่อน Virality (Berger)

1. Social Currency — แชร์แล้วดูฉลาด/เท่
2. Triggers — สิ่งในชีวิตประจำวันที่ทำให้นึกถึง
3. Emotion — โกรธ/ตื่นเต้น/ซึ้ง > เฉย ๆ
4. Public — เห็นคนอื่นใช้/ทำได้ง่าย
5. Practical Value — ใช้ได้จริง
6. Stories — มีโครงเรื่อง ไม่ใช่ list

// FIELD CASE · ICE BUCKET CHALLENGE

K = ~2.4 · บริจาค $115M ใน 8 สัปดาห์

ทำไมไวรัล? ครบ 6 driver: Social Currency (ทำเพื่อเหตุผลใหญ่) + Trigger (วิดีโอเพื่อนเข้าฟีดทุกวัน) + Emotion (สนุก) + Public (ทำได้ทุกคน) + Practical (ช่วยจริง) + Stories (challenge รุ่นต่อรุ่น)

organic reach > $50M media buy
