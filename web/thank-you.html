📋 วิธีใช้ Unlock Code (ทำตามทีละขั้น)
  1. กดปุ่ม "คัดลอก Unlock Code" ด้านบน หรือกดที่ตัวอักษร Code ด้านบนได้เลย — ระบบจะคัดลอกให้อัตโนมัติ
  2. กดปุ่ม "เริ่มอ่านเลย" เพื่อไปหน้าหนังสือ จะเปิดหน้าหนังสือ — บทนำอ่านได้ฟรี ส่วนบทเต็มยังล็อกอยู่
  3. เลื่อนลงไปด้านล่างสุดของหน้า จะเจอกล่องที่เขียนว่า "มี Unlock Code แล้ว?" พร้อมช่องกรอกรหัส
  4. กดที่ช่องกรอก แล้ว "วาง" (Paste) รหัสลงไป บนมือถือ: กดค้างที่ช่อง → เลือก "วาง"  |  บนคอม: กด Ctrl+V (Windows) หรือ ⌘+V (Mac)
  5. กดปุ่ม "ปลดล็อก" รอสักครู่ — เมื่อรหัสถูกต้อง บทเต็มทั้งหมดจะแสดงขึ้นทันที 🎉
  6. เสร็จแล้ว! อ่านได้ทุกบทตลอดชีพ ระบบจะจำรหัสไว้ในเครื่อง — ครั้งต่อไปเปิดมาอ่านได้เลยไม่ต้องใส่ใหม่
    📱 Code ผูกกับอุปกรณ์ 1 เครื่อง — หากต้องการย้ายเครื่อง ให้ใส่ Code อีกครั้งบนเครื่องใหม่

Post-Purchase Rationalization + Endowment Effect — ตอนนี้สมองของคุณกำลังทำสิ่งที่น่าสนใจ: มันกำลัง «ยืนยันกับตัวเอง» ว่าการซื้อครั้งนี้เป็นการตัดสินใจที่ดี — เพราะความรู้สึก «เป็นเจ้าของ» ทำให้คุณให้คุณค่ากับสิ่งที่ได้มามากกว่าก่อนซื้อ (Endowment Effect) และหน้านี้ถูกออกแบบมาเพื่อเสริมความรู้สึกนั้น — ทุก bullet ที่คุณเพิ่งอ่านทำให้รู้สึกว่า «ได้เยอะ» ป้าย «อ่านได้ตลอดชีพ» ลบความกังวลที่เหลือ — นี่คือบทที่ 7 และ 9 ที่ทำงานอยู่ตรงนี้ ก่อนคุณจะเปิดอ่านด้วยซ้ำ
